# User Guides

Role-by-role guides to using and running the Ashley & Hazel wedding portal.

| Guide | For | Covers |
|-------|-----|--------|
| [Guest](GUEST_GUIDE.md) | Invited guests | Logging in, RSVP + meal selection, schedule, blessings, Dancefloor (requests, reactions, now-playing), gallery, notifications, Stag/Hen (if flagged), feedback |
| [Coordinator](COORDINATOR_GUIDE.md) | Helpers with admin access | All planning modules (drag-and-drop Timeline, Menu builder, Feedback triage, real in-app comms) except invite/party management |
| [Couple](COUPLE_GUIDE.md) | Ashley & Hazel | Full access — individual invite identity, invitations, Stag/Hen surprise mechanics, the go-live (phase) flow |
| [IT / Admin](IT_ADMIN_GUIDE.md) | Whoever runs the hosting | Topology, deploy, backups, ops, troubleshooting |
