# User Guides

Role-by-role guides to using and running the Ashley & Hazel wedding portal.

| Guide | For | Covers |
|-------|-----|--------|
| [Guest](GUEST_GUIDE.md) | Invited guests | Logging in with a code, RSVP, schedule, blessings, Dancefloor song requests, gallery |
| [Coordinator](COORDINATOR_GUIDE.md) | Helpers with admin access | All planning modules (except issuing invite codes) |
| [Couple](COUPLE_GUIDE.md) | Ashley & Hazel | Full access — invitations + the go-live (phase) flow |
| [IT / Admin](IT_ADMIN_GUIDE.md) | Whoever runs the hosting | Topology, deploy, backups, ops, troubleshooting |
