# Ashley & Hazel Wedding Portal Prototype

Static clickable prototype for the wedding website and planning dashboard concept.

## How to view
Open `index.html` in a browser.

## Included screens
- Public landing / invite-code gate
- Guest home
- RSVP flow
- Story-led About Us
- Old-web profile cards
- Dancefloor music requests
- Blessings wall
- Calm private dashboard
- Budget tracker
- Planning board
- Contacts

## Accepted concept rules represented
- Deep royal purple + sunflower yellow
- Shona/Jamaican influence expressed visually and emotionally
- Blessings language
- Old-web nostalgia without glitter
- Guest-only RSVP, profiles, music and wall
- Household invite-code model
- Honest accountant-friendly budget dashboard

## Canonical status
This folder is now the canonical local Git repo for the Wedding Dashboard prototype baseline. Use synthetic data only until the data boundary in docs/privacy/DATA_BOUNDARY.md is approved for real wedding data.
